PROJECT 01

Kyobo Bookstore

교보문고 도서 웹페이지

color

“Kyobo Bookstore” 교보문고 웹페이지입니다. 포인트컬러는 도서웹페이지에 맞게 조용하게 침착한인상을 주는 진한 블루 컬러를 사용하였습니다. 중간 구간에는 백그라운드 컬러를 백색으로 설정하여 도서가 돋보일 수 있도록 표현하였습니다.

페이지에 있는 도서들은 브라우저와 서버사이에서 오고가는 데이터의 형식인 JSON을 사용하여 카카오 open api로 가져와 AJAX 하였습니다. 도서에 마우스를 올렸을때 도서에대한 설명이 나타나 어떤 책인지 볼 수 있게 하였습니다.

PROJECT 02

TOYOTA

토요타 한국자동차 웹페이지

color

“Toyota” 한국토요타자동차 웹페이지입니다. 모노톤으로 세련미를 주었습니다.

제품사진이 돋보일 수 있도록 절제된 레이아웃을 사용했습니다.

반응형 웹을 적용하여 여러 디바이스에서 보더라도 편하게 볼 수 있도록 하였습니다. 상품에 대해 좀더 자세히 알아보기 쉽도록 상품사진 아래에 VIEW라는 버튼을 만들었습니다.

PROJECT 03

National Geographic

네셔널지오그래픽 웹페이지

color

“National Geographic” 세계적 명성의 다큐멘터리 잡지인 네셔널지오그래픽 웹페이지입니다. 주제의 분위기와 배경에 맞게 열정이 가득하고 쾌활한 느낌을 주는 노랑컬러와 세련미와 힘을 상징하는 검정컬러를 사용하였습니다.

페이지 구성은 원페이지 형식을 이용하여 움직임을 줄이고 한눈에 볼 수 있도록 하였습니다. 아이콘을 사용하여 간결하면서도 가독성이 높아지도록 하였습니다. 디스플레이 종류에 따라 화면의 크기가 자동으로 최적화 되도록 반응형 웹을 적용하였습니다.

PROJECT 04

Mongchon side dishes

20년 전통 가정식 프리미엄 반찬쇼핑몰 몽촌 반찬 웹페이지

color

“Mongchon side dishes” 20년 전통 가정식 프리미엄 반찬쇼핑몰 몽촌 반찬 웹페이지 입니다. 쇼핑몰의 특성에 맞게 제품이 잘 보이도록 전체적으로 심플한 흰색 컬러를 사용하였습니다.

네비게이션 메뉴를 이동할 때 폰트 컬러가 레드컬러로 변하며 사용자가 현재 위치를 알기 쉽도록 표현하였습니다.

제품 사진에 마우스를 올렸을때 사진에 효과를 주어 사용자가 선택하고 있는것이 무엇인지를 알 수 있도록 하였습니다.
